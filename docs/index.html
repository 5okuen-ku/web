京都大学五億円同好会とは

五億円が好きな人が集うサークルです。
現在、京都大学五億円同好会では新しい会員や五億円を募集しています。

活動内容

五億円に関する情報を収集・発信しています。詳しい内容とかは決まっていないので会員の自由な発想でどんどん活動しましょう。

入会条件

入会の意思があればどなたでも入っていただくことができます。

五億円について

ここでは仮に全部一万円札で五億円を用意したと仮定します。
1万円札は縦76mm、横160mm、厚み0.1mm、重さ1グラム程度 (出典:国立印刷局(大きさ、 厚さ、重さ)) ですので、縦に積み上げると5m、横に広げると608平方メートル、体積は60.8L、重さは50kgになります。
